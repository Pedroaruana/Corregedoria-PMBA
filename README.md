# Sistema de Registro de Ocorrências — COPPM/BA

Sistema web para registro e gerenciamento de ocorrências policiais da Corregedoria da Polícia Militar da Bahia.

## Tecnologias

- React + TypeScript
- Tailwind CSS
- Vite
- Node.js + Express (backend)
- PostgreSQL + Prisma

## Funcionalidades

- Autenticação por matrícula policial
- Registro de Auto de Resistência
- Gerenciamento de ocorrências
- Assinatura digital de termos
- Dashboard com estatísticas

## Como rodar localmente

```bash
npm install
npm run dev
```
